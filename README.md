# AWS Portfolio Website
## Project Name
Personal Portfolio Website using AWS

## Description
This project is a basic personal portfolio website developed using HTML,CSS ,and JavaScript.  
As a 2nd year B.Tech Information Technology student, this project helped me understand how to host a website on cloud using AWS services.

## AWS Service Used
- Amazon S3 (Static Website Hosting)

## Steps to Host Website on AWS S3
1.Created an S3 bucket in AWS
2.Uploaded the HTML, CSS, and JavaScript files
3.Enabled static website hosting in bucket properties
4.Provided public access using bucket policy
5.Accessed the website using the S3 static website endpoint

## Outcome
Through this project, I successfully hosted my portfolio website on AWS and gained basic knowledge about cloud hosting and AWS S3.
